# T-Библиотека (C++ версия)

Консольное приложение для управления личной библиотекой. Позволяет добавлять, просматривать, искать, редактировать и удалять книги, сохраняя данные в текстовый файл.

## Возможности

- **Добавление книги** – ввод названия, автора, жанра, года издания и описания.
- **Просмотр книг** – вывод списка с возможностью сортировки (по названию, автору, году) и фильтрации (по жанру, по статусу прочитана/не прочитана).
- **Избранное** – добавление/удаление книги в/из избранного.
- **Изменение статуса прочитана/не прочитана** – отметка книги как прочитанной или непрочитанной.
- **Список избранных книг** – вывод всех книг, помеченных как избранные.
- **Удаление книги** – удаление выбранной книги с подтверждением.
- **Поиск** – поиск по ключевому слову в названии, авторе, жанре или описании.
- **Сохранение и загрузка** – все данные автоматически сохраняются в файл `lib.txt` при каждом изменении и загружаются при запуске.

## Требования

- Компилятор с поддержкой C++11 и выше (например, g++, clang++, Visual C++).
- Операционная система: Windows, Linux, macOS.

## Сборка и запуск

### Linux / macOS

```bash
g++ -std=c++11 -o tlibrary main.cpp
./tlibrary
```
